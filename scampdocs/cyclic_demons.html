6.6 Cyclic Demons

This cyclic cellular automaton was popularized by Dewdney in his "Computer Recreations" column in Scientific American [MagMach]. A cell in state n of N changes to state n+1 if one of its neighbours is in that state. State N becomes state 0. Initially, each cell is in a uniformly distributed random state. The "order" parameter gives the number of states and "neigbours" chooses between four or eight neighbours when set to 0 or 1. Eventually, the initial granular noise gives way to "droplets" of colour and finally spiral formations or "demons".
