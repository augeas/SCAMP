5.6 Cyclic 1-D

In cyclic cellular automata cells can only switch states in sequence. A cell in state N takes the state N+1 if it has a neighbour in state N+1. The highest state changes to the first state. The "order" parameter gives the number of states, the cells' initial states are chosen randomly, ignoring the seed settings. Values of four [PrimordialCyclic] (and sometimes 5) can produce aesthetic results, that look almost, but not entirely unlike a Bridget Riley. If the "neighbours" parameter is set to 1 then one can "cheat" by also considering next-nearest neighbours.
