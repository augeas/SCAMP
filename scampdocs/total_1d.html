5.3 Totalistic 1-D

Consider a one-dimensional cellular automata with some number of states N. If we consider only the East and West neighbours, that gives N3 combinations and NN3 possible rules. This means that there number of 4-state nearest-neighbour automata is:

340282366920938463463374607431768211456!

With this in mind, Wolfram introduces totalistic cellular automata based on the sum of the states of a cell and its nearest-neighbours (!cite(WolframTotal)]. The "Totalistic 1-D" takes two parameters, the "order" or number of states and the "rule". If there are N states from 0 to N-1, the total cannot be greater than 3N-3, although some values less than this may not be possible. A string of digits base-N is used to decide the next state of each cell, by using the total n to select the nth digit. The base-N number is converted to decimal to produce the rule. SCAMP defaults to order 3 number 65, which starts out fairly interesting, but eventually becomes repetitive.
