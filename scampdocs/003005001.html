3.5.1 One-Dimensional Seeds

3.5.1.1 Single Defects

"Number" gives the number of cells that will be set to 1. "Separation", if non-zero, gives the number of vacant cells between them, otherwise they are equally spaced as far apart as possible. If "justify" has the default value of 0 then the array of 1s is centered, 1 shifts it to the left and 2 to the right.

3.5.1.2 Alternating Blocks

This produces dashed lines of cells in state 1. "On" gives the length of each dash and "off" the size of the gaps between them. "Number" sets the number of dashes, if set to 0 then the line spans the entire array.

3.5.1.3 Random

This produces a random centered band of 1s and 0s. "Probability" is the chance of a cell being set to state 1 out of 1000, "width" sets the width of the band, which spans the array if set to 0.
